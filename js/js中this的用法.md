#js中this的用法
##
this是js语言的一个关键字。它是函数运行时，在函数体内部自动生成的对象，只能在函数体内部使用。  

> function (){  
>   this.x = 1  
>   }  

上面代码中，函数this运行时，内部会自动有一个this对象可以使用。  
在不同的使用场合，this有不同的值。总的来说，this就是函数运行时所在的环境对象。
##情况一：纯粹的函数调用
这是函数的最通常用法，属于全局性调用，因此this就代表全局对象。请看下面这段代码，它的运行结果是1。  

>var x = 1;  
function test() {  
   console.log(this.x);  
}  
test();  // 1  

##情况二：作为对象方法的调用,这时this就指这个上级对象。
##情况三：作为构造方法时，this指向刚刚构造的这个对象
##情况四：apply（）方法是改变改变函数的调用对象。它的第一个参数就表示改变后的调用这个函数的对象。因此，这时this指的就是这第一个参数。
